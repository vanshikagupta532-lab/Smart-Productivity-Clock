# Smart Productivity Clock 🌍⏰

A Python desktop productivity application built using Tkinter.

This application combines a digital clock with productivity tools such as a Pomodoro timer, task tracker, world time zone selection, and desktop notifications.

The goal of this project was to build a simple and useful desktop application while improving Python programming and GUI development skills.

---

## 🚀 Features

### 🕒 Live Digital Clock

* Displays the current time.
* Updates automatically every second.

### 📅 Day and Date

* Shows the current day.
* Shows the current date.

### 🌍 Continent-Based Time Zones

Select a continent and choose a time zone.

Supported continents:

* 🌏 Asia
* 🌍 Europe
* 🌎 North America
* 🌎 South America
* 🌍 Africa
* 🌏 Australia/Oceania

### 🌙 Dark / Light Mode

* Switch between dark mode and light mode.

### ⏰ 12-Hour / 24-Hour Format

* Toggle between 12-hour and 24-hour time formats.

### 📝 Current Task Section

* Enter your current task or activity.

Examples:

* Study Python
* AutoCAD Practice
* PCB Design
* GitHub Project

### 🍅 Pomodoro Timer

Focus timer with:

* ▶️ 25-minute work session
* ☕ 5-minute break session
* ⏹️ Reset button

### 🔔 Desktop Notifications

A desktop notification appears when a timer session finishes.

## ⚙️ Installation

Install the required packages:

```bash
pip install -r requirements.txt
```

Or install them manually:

```bash
pip install pytz
pip install plyer
```

---

## ▶️ Run the Application

Open the terminal and run:

```bash
python main.py
```

or

```bash
py main.py
```
